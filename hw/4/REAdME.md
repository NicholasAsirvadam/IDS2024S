NAND Equivalence  

$$
\begin{eqnarray}
&& (A \uparrow A) \uparrow ( ((A \uparrow B) \uparrow (A \uparrow B)) \uparrow ((A \uparrow B) \uparrow (A \uparrow B)) ) ~, \nonumber \\
&=& (A \uparrow A) \uparrow ( AB \uparrow AB ) ~, \nonumber \\
&=& (A + AB)  ~, \nonumber \\
&=& A ~, \nonumber
\end{eqnarray}
$$
